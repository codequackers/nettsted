Programmering i nettleseren

Det går an å skrive JavaScript rett inn i nettleseren. Det kan være greit for å teste enkle ting, men egner seg ikke til større programmer. For å få dette til må vi finne konsollen i nettleseren. Hvordan vi finner den avhenger litt av hvilken nettleser du bruker.


Firefox

I Firefox kan du bruke hurtigtastene Ctrl + Shift + K på Windows, eller Command + Option + K på Mac. Eller du kan høyreklikke på en nettside og velge Inspiser Element. Finn console i vinduet som åpnes. Hurtig link for åpning av consolen

Chrome

I Chrome kan du bruke hurtigtastene Ctrl + Shift + J og finn console i vinduet som åpnes, eller Command + Option + J på Mac. Du kan også høyreklikke og velge Inspiser og så finne console i vinduet som åpnes.

Safari

For å få tilgang til konsollen i Safari må du først aktivere utviklerverktøyene. Hvis du ikke ser Utvikle-menyen i menylinjen går du på Safari > Valg og klikker på Avansert. Under denne fanen markerer du «Vis Utvikle-menyen i menylinjen». Når det er gjort får du tilgang til konsollen på en av følgende måter.

  • Klikk på Utvikle > Vis JavaScript Konsoll
  • Hurtigtast Command + Option + C
  • Høyreklikk på nettsiden og velg Inspiser element og finn så Konsoll i vinduet som dukker opp

Test consolen

Når du har åpnet konsollen så kan du teste enkel programkode. For eksempel kan JavaScript brukes som en ganske enkel kalkulator. Prøv å skriv inn noen enkle regnestykker og trykk Enter. For eksempel 2+2 eller 5+3*2 og se hva du får. Du kan også prøve å kopiere koden under, lime den inn i konsollen og trykke Enter.
